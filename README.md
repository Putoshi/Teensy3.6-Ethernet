### Wiring

|  Teensy 3.6 Pins  |  WIZ850io Pins  |
| ---- | ---- |
|  10 – SCS  |  SCNn  |
|  11 – MOSI  |  MOSI  |
|  12 – MISO  |  MISO  |
|  13 – SCLK  |  SCLK  |
|  GND  |  GND  |
|  3.3V  |  3.3V  |
